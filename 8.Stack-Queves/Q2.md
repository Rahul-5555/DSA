# Implement Stack Using One Queue (LeetCode 225)

This project demonstrates how to implement a **Stack (LIFO)** using only **one Queue (FIFO)**.

---

## ðŸ”¹ Stack vs Queue

- **Stack**: Last In First Out (LIFO)
- **Queue**: First In First Out (FIFO)

---

## ðŸ”¹ Approach Used

We use **one queue** and rotate its elements during `pop()` and `top()` operations so that the **last inserted element** comes to the front.

---

## ðŸ”¹ Code Implementation

```js
var MyStack = function () {
  this.q = [];
}

MyStack.prototype.push = function (x) {
  this.q.push(x);
};

MyStack.prototype.pop = function () {
  let n = this.q.length;
  for (let i = 0; i < n - 1; i++) {
    this.q.push(this.q.shift());
  }
  return this.q.shift();
};

MyStack.prototype.top = function () {
  let n = this.q.length;
  for (let i = 0; i < n - 1; i++) {
    this.q.push(this.q.shift());
  }
  let top = this.q[0];
  this.q.push(this.q.shift());
  return top;
};

MyStack.prototype.empty = function () {
  return this.q.length === 0;
};
