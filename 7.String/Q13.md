## ðŸ” Step-by-Step Logic

1. Agar `s.length !== t.length` â†’ direct `false`
2. Do empty objects banao:

   * `mapST = {}`

3. Strings par loop chalao (`i = 0 â†’ n-1`)
4. Har index par:

   * Agar `mapST[s[i]]` pehle se exist karta hai **aur** `t[i]` se match nahi karta â†’ `false`

5. Agar koi conflict nahi mila:

   * `mapST[s[i]] = t[i]`
   * `mapTS[t[i]] = s[i]`
6. Loop complete hone ke baad â†’ `true`
